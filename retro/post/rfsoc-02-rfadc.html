RFSoC学习 03 - RF-ADC 架构
2006-02-14|RFSoC|RFSoCFPGAXilinx|⏱ 9 min | 2704 字

RF-ADC Tile类型

RF-ADC有两种tile类型：Dual RF-ADC和Quad RF-ADC。可用的tile类型取决于具体器件（参见DS889）。

每个tile包含一个PLL和时钟电路。tile内的所有RF-ADC共享这个公共时钟基础设施。

Quad RF-ADC Tile

Quad RF-ADC tile由4个RF-ADC组成，排列为两对。每个转换器可以单独配置为实信号输入，或作为一对配置为I/Q输入信号。

Gen 1/Gen 2架构：


VIN_P/VIN_N → BUF → ADC → I/Q分离 → NCO/混频器 → DDC → FIFO → AXI4-Stream

每个通道：

  • 差分输入缓冲器
  • 12位RF-ADC
  • I/Q分离（0°/90°）
  • NCO和复数混频器
  • DDC（数字下变频器）

Gen 3/DFE架构：


VIN → ADC → DSA → DDC → FIFO → AXI4-Stream

每个通道：

  • 片上校准的50Ω终端
  • 14位RF-ADC
  • DSA（数字步进衰减器）
  • DDC
  • FIFO

Dual RF-ADC Tile

Dual RF-ADC tile由2个RF-ADC组成。这些转换器可以单独配置为实信号输入，或作为一对配置为I/Q输入信号。

Gen 1/Gen 2架构：


VIN_P/VIN_N (100Ω) → BUF → RF-ADC (4GS/s, 12-bit) → I/Q分离 → NCO → DDC → FIFO

Gen 3/DFE架构：


VIN → RF-ADC → DSA → DDC → FIFO → AXI4-Stream

RF-ADC特性

Tile配置

  • 每个tile包含4个或2个RF-ADC和1个PLL
  • Gen 1/Gen 2：12位ADC分辨率，16位DSP数据路径。每个12位数据流在RF-ADC核心输出后MSB对齐到16位样本，然后传递到DDC模块
  • Gen 3/DFE：14位ADC分辨率，16位DSP数据路径。每个14位数据流在RF-ADC核心输出后MSB对齐到16位样本，然后传递到DDC模块
  • 每个Tile包含4个（Quad）或2个（Dual）通道。采样率取决于具体器件，详见DS926

抽取滤波器

  • Gen 1/Gen 2：1x（旁路滤波器）、2x、4x、8x
  • Gen 3/DFE：1x（旁路滤波器）、2x、3x、4x、5x、6x、8x、10x、12x、16x、20x、24x、40x
  • 抽取滤波器的通带为奈奎斯特带宽的80%，阻带衰减89dB

数字复数混频器

  • 全复数混频器支持来自RF-ADC的实信号或I/Q输入
  • 每个RF-ADC有48位数控振荡器（NCO）
  • 固定Fs/4、Fs/2低功耗频率混频模式，其中Fs是采样频率
  • 支持I/Q和实信号输入

单频段/多频段灵活性

  • 每对RF-ADC支持2个频段
  • 每个Quad RF-ADC tile支持4个频段
  • 可配置为实信号或I/Q输入

其他特性

  • 旁路模式下可访问RF-ADC的全带宽
  • 输入信号幅度阈值：每个RF-ADC有2个可编程阈值标志
  • 内置数字校正用于外部模拟正交调制器：
    • 支持I/Q输入对（两个RF-ADC）的增益、相位和偏移校正
  • SYSREF输入信号用于多通道同步
  • 灵活的AXI4-Stream接口支持广泛的可编程逻辑时钟速率和转换器采样率
  • 模拟输入：每个RF-ADC有专用的差分输入缓冲器，带片上校准的100Ω终端；提供输出共模参考电压（VCM），用于DC耦合前端电路
  • Gen 3/DFE：数字步进衰减器（DSA）
  • Gen 3/DFE：时分双工（TDD）应用中的省电模式
  • Gen 3/DFE：TDD应用中RX和观测通道的不同抽取因子和FIFO数据速率

数据路径详解

Gen 1/Gen 2数据路径


模拟输入 → 输入缓冲器 → 12位ADC → MSB对齐到16位 → DDC → FIFO → AXI4-Stream

（MSB：将有效数据位靠左放置，低位补0，方便后续DSP处理）

DDC模块：

  • 抽取滤波器：1x/2x/4x/8x
  • NCO：48位，生成本振信号
  • 复数混频器：将RF信号下变频到基带
  • I/Q分离：输出I路和Q路

时钟关系：

  • T1：采样时钟
  • T2：T1/2
  • T4：T1/4
  • T8：T1/8

抽取后的数据速率 = 采样率 / 抽取因子

Gen 3/DFE数据路径


模拟输入 → 50Ω终端 → 14位ADC → DSA → MSB对齐到16位 → DDC → FIFO → AXI4-Stream

DSA（数字步进衰减器）：

  • 可编程衰减
  • 用于AGC（自动增益控制）
  • 实时控制接口

扩展的抽取率：

  • 1x到40x的灵活配置
  • 支持更多带宽组合
  • 例如：4GSPS采样 + 40x抽取 = 100MSPS输出

工作模式

实信号模式

单个ADC处理一路实信号：


VIN → ADC → DDC → 实信号输出

I/Q模式

两个ADC组成一对，处理正交信号：


VIN_I → ADC0 → DDC → I路输出
VIN_Q → ADC1 → DDC → Q路输出

I/Q模式用于：

  • 复数调制信号
  • 零中频（Zero-IF）架构
  • 正交解调

多频段模式

每对RF-ADC可以同时处理2个频段：


RF输入 → ADC → DDC0 (频段1) → 输出0
              → DDC1 (频段2) → 输出1

每个DDC有独立的NCO，可以下变频到不同的中心频率。

校准和信号检测

校准冻结（Gen 3/DFE）

时间交织RF-ADC依赖后台校准来最大化性能。校准算法需要输入信号存在一段时间才能收敛。当输入信号不存在时，校准会发散。

校准冻结机制：

  • adcXY_int_cal_freeze：PL发出的信号，指示IP冻结校准
  • adcXY_cal_frozen：校准已冻结时置位
  • 通常在RF-ADC输出低于某个阈值时触发

信号幅度检测器（Gen 3/DFE）

  • adcXY_sig_detect：信号幅度检测器的状态输出
  • 异步信号
  • 用于检测输入信号是否存在

接口

AXI4-Stream数据接口

每个RF-ADC通道有一个AXI4-Stream主接口：

  • mXY_axis_tdata：数据
  • mXY_axis_tvalid：数据有效
  • mXY_axis_tready：准备接收

其中X是tile编号，Y是通道编号（0-3）。

实时DSA接口（Gen 3/DFE Dual RF-ADC）

  • adcX_ZZ_dsa_code[4:0]：RF-ADC衰减码（输入）
  • adcX_dsa_update：置位以锁存衰减码到RF-ADC（输入）

时钟域：s_axi_aclk

其中ZZ是01（tile中的下方RF-ADC）或23（tile中的上方RF-ADC）。

阈值检测

每个RF-ADC有2个可编程阈值检测器：

  • 阈值0：低阈值
  • 阈值1：高阈值

用途：

  • 自动增益控制（AGC）
  • 信号存在检测
  • 过载检测

正交调制校正（QMC）

用于补偿外部模拟正交调制器的不完美：

  • 增益不平衡：I路和Q路幅度不一致
  • 相位不平衡：I路和Q路相位差不是精确90°
  • DC偏移：I路和Q路的直流偏移

QMC在数字域进行校正，支持I/Q输入对（两个RF-ADC）。
