名词解释

基础

RFSoC (RF System on Chip)

  • 集成了RF数据转换器的Zynq UltraScale+ SoC
  • 包含PS（Processing System）、PL（Programmable Logic）和RF Data Converters

PS (Processing System)

  • Arm处理器子系统：Quad-core Cortex-A53 + Dual-core Cortex-R5
  • 运行操作系统和应用软件

PL (Programmable Logic)

  • FPGA可编程逻辑部分
  • 实现数字信号处理和自定义硬件加速

Tile

  • RF Data Converter 的基本组织单元
  • 包含2个或4个转换器 + 1个PLL + 时钟电路
  • 每个tile内的转换器共享时钟基础设施

Channel / Block / Slice

  • 三个术语在文档中同义
  • 指tile中的单个RF-ADC或RF-DAC

RF Data Converter

RF-ADC (RF Analog-to-Digital Converter)

  • 射频模数转换器
  • 直接对RF信号采样，无需下变频到中频

RF-DAC (RF Digital-to-Analog Converter)

  • 射频数模转换器
  • 直接生成RF信号，无需上变频

Quad Tile

  • 包含4个转换器的tile
  • Quad RF-ADC：每个ADC有4倍交织
  • Quad RF-DAC：每个DAC有1个DUC

Dual Tile

  • 包含2个转换器的tile
  • Dual RF-ADC：每个ADC有8倍交织，采样率是Quad的2倍
  • Dual RF-DAC（Gen 3/DFE）：每个DAC有2个DUC，支持双频段

Sub-ADC

  • 构成RF-ADC的子转换器
  • Quad RF-ADC：每个ADC由4个sub-ADC组成
  • Dual RF-ADC：每个ADC由8个sub-ADC组成

Interleaving Factor（交织因子）

  • 时间交织技术的倍数
  • 多个低速ADC轮流采样，等效合成一个高速ADC
  • 交织因子越高，采样率越高

DSP

DDC (Digital Down Converter)

  • RF-ADC中的数字信号处理模块
  • 包含：抽取滤波器 + NCO + 混频器
  • 将RF信号下变频到基带

DUC (Digital Up Converter)

  • RF-DAC中的数字信号处理模块
  • 包含：插值滤波器 + NCO + 混频器
  • 将基带信号上变频到RF

NCO (Numerically Controlled Oscillator，数控振荡器)

  • 48位数字振荡器
  • 生成数字本振信号
  • 用于内部混频操作

Decimation（抽取）

  • 降低数据速率的过程
  • 例如：4GSPS采样 + 8x抽取 = 500MSPS输出
  • Gen 1/2：1x, 2x, 4x, 8x
  • Gen 3/DFE：1x~40x

Interpolation（插值）

  • 提高数据速率的过程
  • 例如：100MSPS基带 + 8x插值 = 800MSPS输出
  • Gen 1/2：1x, 2x, 4x, 8x
  • Gen 3/DFE：1x~40x

Mixer（混频器）

  • 本质是复数乘法器
  • 信号频率搬移
  • 与NCO配合实现频率变换

信号类型

Real Signal（实信号）

  • 单路实数信号
  • 单个ADC/DAC处理

I/Q Signal（正交信号）

  • I（In-phase，同相）+ Q（Quadrature，正交）
  • 两个ADC/DAC组成一对
  • 相位差90°
  • 用于复数调制

Complex Signal（复信号）

  • 用I/Q表示的信号
  • 复数形式：I + jQ
  • 包含幅度和相位信息

时钟术语

Tn Clock

  • T1：根时钟（通常是采样时钟）
  • Tn：T1除以n的时钟
  • 例如：T8 = T1/8

Fs (Sampling Frequency)

  • 采样频率
  • Fs/2：奈奎斯特频率
  • Fs/4：四分之一采样频率

PLL (Phase-Locked Loop，锁相环）

  • 每个tile有一个PLL
  • 可以从外部参考时钟倍频
  • 生成采样时钟

SYSREF (System Reference)

  • 系统参考信号
  • 用于多tile同步
  • 同步时钟分频器和FIFO

同步

MTS (Multi-Tile Synchronization，多Tile同步）

  • 实现多个tile之间的精确对齐
  • 消除延迟不确定性
  • 关键应用：MIMO、相控阵

Latency（延迟）

  • 信号通过系统的时间延迟
  • MTS需要匹配所有tile的延迟

FIFO

  • 在PL时钟域和采样时钟域之间传输数据
  • 引入非确定性延迟，需要MTS校正

Deterministic Latency（确定性延迟）

  • 固定的、可预测的延迟
  • 通过Target_Latency机制实现

校正和控制

QMC (Quadrature Modulator Correction，正交调制校正）

  • 补偿I/Q不平衡
  • 校正增益不平衡、相位不平衡、DC偏移

DSA (Digital Step Attenuator，数字步进衰减器）

  • Gen 3/DFE的RF-ADC特性
  • 可编程衰减
  • 用于AGC（自动增益控制）

VOP (Variable Output Power，可变输出功率）

  • Gen 3/DFE的RF-DAC特性
  • 全范围电流控制
  • 精细调节输出功率

Sinc Correction

  • 补偿DAC的sinc响应
  • Gen 1/2：第一奈奎斯特区
  • Gen 3/DFE：第一和第二奈奎斯特区

AGC (Automatic Gain Control，自动增益控制）

  • 自动调整信号增益
  • 使用阈值检测器和DSA实现

接口

AXI4-Stream

  • 数据流接口
  • 连接RF转换器和PL逻辑
  • 包含TDATA、TVALID、TREADY信号

AXI4-Lite

  • 控制接口
  • 访问配置寄存器
  • 读取状态信息

CML (Current Mode Logic)

  • 电流模式逻辑
  • 时钟输入缓冲器类型
  • 100Ω片上终端

代次和器件

Gen 1

  • 器件型号：XCZU2xDR
  • 12位ADC
  • 基础功能

Gen 2

  • 器件型号：XCZU39DR
  • 12位ADC
  • 仅Quad RF-ADC tile

Gen 3

  • 器件型号：XCZU4xDR
  • 14位ADC
  • 扩展的抽取/插值率（1x~40x）
  • DSA、VOP、TDD模式

DFE (Digital Front End)

  • 器件型号：XCZU6xDR
  • 针对数字前端应用优化
  • 与Gen 3类似的特性

应用相关

TDD (Time Division Duplexing，时分双工）

  • 收发时分复用
  • Gen 3/DFE支持省电模式
  • RX/Obs通道共享

MIMO (Multiple-Input Multiple-Output)

  • 多输入多输出
  • 需要多通道同步
  • 典型应用：5G基站

Massive MIMO

  • 大规模MIMO
  • 64/128/256天线阵列
  • 需要MTS实现相位对齐

Beamforming（波束成形）

  • 通过相位控制实现定向发射/接收
  • 需要精确的通道同步

Carrier Aggregation（载波聚合）

  • 多个频段同时使用
  • 提高带宽
  • Dual RF-DAC的双DUC支持

频率相关

Nyquist Zone（奈奎斯特区）

  • 第一奈奎斯特区：DC到Fs/2
  • 第二奈奎斯特区：Fs/2到Fs
  • 更高奈奎斯特区：依次类推

Direct RF Sampling（直接RF采样）

  • ADC/DAC直接工作在RF频段
  • 无需中频链路
  • 简化模拟设计

Zero-IF（零中频）

  • 直接将RF信号下变频到基带
  • 使用I/Q架构
  • 无中频阶段
