RF-DAC Tile类型

RF-DAC有两种tile类型：Quad RF-DAC和Dual RF-DAC。可用的tile类型取决于具体器件（详见DS889）。

Gen 1/Gen 2器件：只有Quad RF-DAC tile（4个RF-DAC）

Gen 3/DFE器件：同时支持Quad RF-DAC（4个）和Dual RF-DAC（2个）

Quad RF-DAC Tile

Gen 1/Gen 2架构：


AXI4-Stream → DUC → I/Q合成 → NCO/混频器 → DAC → 50Ω终端 → VOUT_P/VOUT_N

每个通道：

  • DUC（数字上变频器）
  • I/Q合成（0°/90°）
  • NCO和复数混频器
  • 14位DAC
  • 50Ω输出终端
  • DAC_AVTT电源

Gen 3/DFE架构：


AXI4-Stream → FIFO → DUC → VOP → DAC → 50Ω终端 → VOUT_P/VOUT_N

每个通道：

  • FIFO
  • DUC
  • VOP（可变输出功率）
  • 14位DAC
  • 50Ω输出终端
  • DAC_AVTT电源

Dual RF-DAC Tile (Gen 3/DFE)


AXI4-Stream → FIFO → DUC0 ─┐
                           ├→ VOP → DAC → 50Ω → VOUT_P/VOUT_N
AXI4-Stream → FIFO → DUC1 ─┘

每个DAC有2个专用DUC，支持双频段应用。

RF-DAC特性

Tile配置

  • Gen 1/Gen 2：每个tile 4个RF-DAC和1个PLL
  • Gen 3/DFE：每个tile 4个或2个RF-DAC和1个PLL
  • 14位RF-DAC分辨率，16位数字信号处理路径；数据MSB对齐到16位
  • 器件相关的采样速度，见DS889

插值滤波器

  • Gen 1/Gen 2：1x（旁路滤波器）、2x、4x、8x
  • Gen 3/DFE：1x（旁路滤波器）、2x、3x、4x、5x、6x、8x、10x、12x、16x、20x、24x、40x；IMR模式下额外提供2x
  • 插值滤波器的通带为奈奎斯特带宽的80%，阻带衰减89dB

数字复数混频器

  • 全复数混频器支持实信号或I/Q输出信号到RF-DAC
  • 每个RF-DAC有48位NCO
  • 固定Fs/4、Fs/2低功耗频率混频模式
  • 支持混合模式，可在第二奈奎斯特区获得更高的输出功率

单频段/多频段灵活性

  • 每对RF-DAC支持2个频段
  • 每个RF-DAC tile支持4个频段
  • 可配置为实信号或I/Q输出

其他特性

  • 旁路模式下的全奈奎斯特带宽
  • 外部模拟正交调制器的数字校正：
    • 支持I/Q输出对（两个RF-DAC）的增益、相位和偏移校正
  • Gen 1/Gen 2：第一奈奎斯特区的Sinc校正
  • Gen 3/DFE：第一和第二奈奎斯特区的Sinc校正
  • 外部输入信号（SYSREF）用于多通道同步
  • 每个tile的电流模式逻辑（CML）时钟输入缓冲器，带片上校准的100Ω终端；提供RF-DAC采样时钟或为片上PLL提供参考时钟（在仅包含Dual RF-DAC的Gen 3/DFE器件中，奇数编号的Tile（Tile 1、Tile 3）没有外部时钟输入，需要从其他Tile获取时钟）
  • Gen 1/Gen 2：支持20mA或32mA输出功率模式
  • Gen 3/DFE：可变输出功率（VOP）支持全范围电流吸收，向后兼容Gen 1和Gen 2的20/32mA模式
  • Gen 3/DFE：时分双工（TDD）应用中各个功能块的省电模式

数据路径详解

Gen 1/Gen 2数据路径


AXI4-Stream → DUC → I/Q合成 → NCO混频 → 14位DAC → 模拟输出

DUC模块：

  • 插值滤波器：1x/2x/4x/8x
  • NCO：48位，生成本振信号
  • 复数混频器：将基带信号上变频到RF
  • I/Q合成：将I路和Q路合成为实信号

时钟关系：

  • T1：采样时钟
  • T2：T1/2
  • T4：T1/4
  • T8：T1/8

插值前的数据速率 = 采样率 / 插值因子

Gen 3/DFE数据路径


AXI4-Stream → FIFO → DUC → VOP → 14位DAC → 50Ω终端 → 模拟输出

VOP（可变输出功率）：

  • 可编程输出电流
  • 全范围调节
  • 向后兼容20/32mA模式

扩展的插值率：

  • 1x到40x的灵活配置
  • IMR模式下额外2x
  • 例如：100MSPS基带 + 40x插值 = 4GSPS输出

Dual RF-DAC的双频段能力（Gen 3/DFE）

每个DAC有2个独立的DUC：


频段1数据 → FIFO → DUC0 ─┐
                        ├→ 数字相加 → VOP → DAC → 双频段输出
频段2数据 → FIFO → DUC1 ─┘

每个DUC有独立的NCO，可以上变频到不同的中心频率。

应用场景：

  • 双频Wi-Fi（2.4GHz + 5GHz）
  • 载波聚合
  • 多标准基站

工作模式

实信号模式

单个DAC输出一路实信号：


基带数据 → DUC → DAC → 实信号输出

I/Q模式

两个DAC组成一对，输出正交信号：


I路数据 → DAC0 → I路输出
Q路数据 → DAC1 → Q路输出

I/Q模式用于：

  • 复数调制信号
  • 零中频（Zero-IF）架构
  • 正交调制

多频段模式

Quad RF-DAC：每对DAC可以同时输出2个频段 Dual RF-DAC：每个DAC可以同时输出2个频段


频段1 → DUC0 ─┐
              ├→ DAC → 多频段输出
频段2 → DUC1 ─┘

Sinc校正

DAC的输出具有sinc响应特性，导致频率响应在奈奎斯特区内不平坦（在高频处滚降）。Sinc校正补偿这种响应。

  • Gen 1/Gen 2：第一奈奎斯特区（DC到Fs/2）的Sinc校正
  • Gen 3/DFE：第一和第二奈奎斯特区（DC到Fs）的Sinc校正

第二奈奎斯特区校正支持混合模式操作，在Fs/2到Fs频段最大化输出功率。

正交调制校正（QMC）

用于补偿外部模拟正交调制器的不完美：

  • 增益不平衡：I路和Q路幅度不一致
  • 相位不平衡：I路和Q路相位差不是精确90°
  • DC偏移：I路和Q路的直流偏移

QMC在数字域进行校正，支持I/Q输出对（两个RF-DAC）。

输出功率控制

Gen 1/Gen 2

两种固定输出功率模式：

  • 20mA模式
  • 32mA模式

Gen 3/DFE

VOP（可变输出功率）：

  • 全范围电流吸收控制
  • 精细调节输出功率
  • 向后兼容20/32mA模式

TDD模式（Gen 3/DFE）

时分双工应用中的省电模式：

  • TDD模式下可以在发射间隙关闭DUC、混频器等数字模块，仅保留FIFO和时钟，实现快速唤醒
  • 在不发射时降低功耗
  • 快速唤醒和关闭

接口

AXI4-Stream数据接口

每个RF-DAC通道有一个AXI4-Stream从接口：

  • sXY_axis_tdata：数据
  • sXY_axis_tvalid：数据有效
  • sXY_axis_tready：准备接收

其中X是tile编号，Y是通道编号（0-3）。

时钟接口

  • dac_clk_p/dac_clk_n：差分时钟输入
  • 100Ω片上校准终端
  • 可以是采样时钟或PLL参考时钟

应用场景

根据PG269文档，RF-DAC的典型应用包括：

  • 多频段、多模式3G、4G和5G蜂窝无线电
  • 多天线系统（Massive MIMO、AAS、AAA）
  • 有线基础设施（DOCSIS 3.x/4.0）
  • 软件定义无线电
  • 微波和毫米波无线电
  • 测试测量应用
